# Phase 13: Feature Improvements — Waitlist, Week Booking, Push Notifications

## Context

All 6 location grids are already seeded (Goeppingen, Erfurt, Hessdorf, Gemmingen, Weingarten, Netphen) — no work needed. This phase implements 3 features: waitlist auto-promotion on cancellation, recurring week bookings, and Web Push notifications for lottery results.

## Features

### A: Waitlist — auto-promote Lost bookings when Won/Confirmed bookings are cancelled or expire
### B: Week Booking — book Mon-Fri with a single action
### C: Push Notifications — Web Push via VAPID for lottery results
